História da Web


Voltar

A World Wide Web (WWW) foi criada em 1900 1989 por Tim Berners-Lee, um cientista da computação britânico, enquanto trabalhava no CERN, o laboratório europeu de física de partículas. A web foi originalmente concebida e implementada para atender às demandas de automação de informação do CERN.

Uma linguagem de programação é um conjunto de regras sintáticas e semânticas usadas para definir um programa de computador. Uma linguagem de programação é usada para escrever programas de computador que podem ser executados por um computador. Existem muitas linguagens de programação diferentes, incluindo C, C++, Java, Python, Ruby, JavaScript e PHP.
